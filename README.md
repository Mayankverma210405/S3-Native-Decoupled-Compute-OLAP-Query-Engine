# S3-Native-Decoupled-Compute-OLAP-Query-Engine

> A cloud-native analytical query engine that executes SQL directly over datasets stored in AWS S3 using DuckDB, following a decoupled compute and storage architecture.

---

## 🚧 Project Status

**Currently Under Active Development**

This repository is being built incrementally with a strong focus on software architecture, cloud engineering, data engineering, and engineering best practices.

---

## 🎯 Project Goals

* Execute SQL queries directly on CSV datasets stored in AWS S3.
* Demonstrate decoupled compute and storage architecture.
* Use DuckDB as an embedded OLAP query engine.
* Implement direct browser-to-S3 uploads using Presigned URLs.
* Build a production-quality engineering portfolio project.

---

## 🛠 Planned Technology Stack

| Layer             | Technology                |
| ----------------- | ------------------------- |
| Frontend          | React + Vite + TypeScript |
| Backend           | FastAPI                   |
| Query Engine      | DuckDB                    |
| Object Storage    | AWS S3                    |
| Metadata Database | PostgreSQL (Supabase)     |
| Deployment        | Vercel + Render           |
